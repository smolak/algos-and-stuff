# Constant O(1)

The number of steps does not depend on the input size.

exmaples:
```js
const foo = (number) => {
  let result = number;
  for (let i = 0; i < 5; i++) {
    result += number;
  }
  
  return result;
}
```

```js
const bar = (numbers) => {
  return numbers[0] + numbers[numbers.length - 1]; 
}
```
